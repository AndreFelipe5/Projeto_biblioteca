Biblioteca LP

Sistema de gerenciamento de biblioteca desenvolvido em Python utilizando arquivos TXT para armazenamento de dados.

Objetivo

O projeto foi criado com o objetivo de praticar conceitos fundamentais de Python, como:

- Funções
- Módulos
- Estruturas condicionais
- Estruturas de repetição
- Manipulação de arquivos
- Listas
- CRUD (Create, Read, Update e Delete)

Funcionalidades

Usuários

- Cadastrar usuários

Livros

- Cadastrar livros
- Editar livros
- Listar livros disponíveis

Empréstimos

- Emprestar livros
- Devolver livros
- Armazenar livros emprestados em arquivo separado

Relatórios

- Visualizar livros disponíveis
- Visualizar livros emprestados

Como Executar

1. Instale o Python 3.
2. Clone ou baixe o projeto.
3. Execute o arquivo principal:

python menu.py

Menu do Sistema

1 - Cadastrar Usuário

2 - Cadastrar Livro

3 - Empréstimo de Livros

4 - Devolução de Livros

5 - Editar Livro

6 - Relatório da Biblioteca

7 - Sair

- Escrita em arquivos TXT
